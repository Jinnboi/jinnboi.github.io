Sorting Algorithm Visualizer

In my Data Structures and Algorithms class, we spent a couple weeks reviewing several popular sorting algorithms and their efficacy with large arrays. We started with the most basic sorts (such as Bubble and Selection sort) and ultimately ended with the faster Quick and Counting sorts. However, one of the issues that my classmates and I had was a lack of visual understanding; that is, how does the sorting algorithm look? To solve this problem, I developed this program in Java (using the Java Swing framework) that displayed all of the sorts that were covered in the lectures. For more information, click on the GIF!

Neural Network

As someone heavily interested in Robotics, I thought that it would be best to learn more about machine learning models. I had done some work with scikit-learn in the past (mostly for SVD and KNN), but I hadn’t actually coded a ML model from scratch. That’s when I decided to create my own Neural Network using only Python and NumPy. This project was unexpectedly challenging because I knew little to no Linear Algebra, but the end result was totally worth it! Using the MNIST data set, my Neural Network consistently achieved 97-98% accuracy! For more information, click the image!
